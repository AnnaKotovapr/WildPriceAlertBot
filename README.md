##  Описание
Этот проект предназначен для автоматического отслеживания цен на товары на сайте Wildberries. 
На данный момент проект находится на этапе планирования. 

Основные функции и модули, которые будут реализованы в будущем:
Функциональные возможности: 
1. Отслеживание изменений цен: - Бот будет автоматически проверять текущие цены на товары, выбранные пользователем, с регулярной периодичностью (например, каждые несколько часов). 
2. Отправка уведомлений пользователям: - Пользователи будут получать уведомления (через Telegram) о снижении или повышении цены на выбранные товары. 
3. Хранение истории цен: - Для каждого товара будет храниться история изменений цен, что позволит пользователям видеть, как цена менялась со временем. 


## Технологии
- Python 3.9.10
- Парсинг данных: Selenium, API Wildberries
- База данных: SQLite или PostgreSQL
- Уведомления: Python-telegram-bot
